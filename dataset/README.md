Dataset Description – Heart Disease Prediction
#Overview

This folder contains the dataset used for building the Heart Disease Prediction AI/ML model. The dataset consists of medical and physiological attributes of patients, which are used to predict the likelihood of heart disease.

The data plays a crucial role in training the machine learning model to identify patterns and risk factors associated with heart conditions.

#Dataset Information

The dataset includes multiple features related to a patient’s health condition. Each row represents an individual patient record, and each column represents a specific medical attribute.

#Key Features

Some of the important attributes in the dataset include:

Age → Age of the patient
Sex → Gender of the patient
Chest Pain Type (cp) → Type of chest pain experienced
Resting Blood Pressure (trestbps)
Cholesterol (chol) → Serum cholesterol level
Fasting Blood Sugar (fbs)
Resting ECG (restecg) → Electrocardiographic results
Max Heart Rate Achieved (thalach)
Exercise Induced Angina (exang)
Oldpeak → ST depression induced by exercise
Slope → Slope of peak exercise ST segment
Number of Major Vessels (ca)
Thalassemia (thal)

-Target Variable
target
0 → No heart disease
1 → Presence of heart disease
-Data Format
File format: CSV / Excel (depending on your files)
Structured tabular dataset
Suitable for supervised machine learning (classification)
